## 📋 À PROPOS DU PROJET

**Huggy Love** est une landing page de Saint-Valentin 2026, conçue pour promouvoir et vendre une peluche unique avec un concept émotionnel puissant : **"L'Étreinte Éternelle"**.

### ✨ **Points clés du projet**
- 🎨 Design moderne et romantique avec animations premium
- 💝 Message émotionnel centré sur l'amour à distance
- 📱 Responsive sur tous les appareils (mobile, tablet, desktop)
- 💳 Formulaire de paiement complet et fonctionnel
- 🎬 Animations Framer Motion fluides et professionnelles
- 📊 Plusieurs sections de conviction (bénéfices, témoignages, réassurance)

## 🛠️ TECHNOLOGIES UTILISÉES

### Frontend Stack
- **React 18** - Framework UI moderne et performant
- **TypeScript** - Typage statique pour code robuste
- **Vite** - Build ultra-rapide et optimisé
- **Tailwind CSS** - Styling utilitaire et thème personnalisé
- **Framer Motion** - Animations fluides et complexes
- **React Router v6** - Navigation déclarative

### Composants & Outils
- **shadcn/ui** - Composants UI accessibles
- **Lucide React** - Icônes modernes
- **Sonner** - Notifications toast professionnelles
- **jsPDF** - Génération de factures PDF
- **html2canvas** - Export d'images
